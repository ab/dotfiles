GPG as SSH agent notes:
- set gpg-agent with SSH_AUTH_SOCK
Adding key to GPG agent for the first time:
- ssh-add foo.key
    - will prompt for existing password, then ask for new wrapping pass
    - key will be stored in ~/.gnupg/private-keys-v1.d/
    - key grip is saved in ~/.gnupg/sshcontrol with comments
